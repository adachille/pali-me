# Issue 004: UX Improvements for MVP

## Overview

Improve the user experience of the MVP by fixing paper-cuts and polishing the UX.

## Paper cuts / bug fixes

- When adding cards to a deck, we shouldn't just be able to choose from existing cards, we should also be able to create new cards
- When on the card creation screen, we should be able to add just that card OR add that card and immediately add a new card
- Diacritics seem to be messing up lexographical ordering (e.g. ā is sorted to the bottom). Can we fix this so that a and ā are equivalent?
- When you enable "endless mode" in the study settings, the modal disappears and then reappears again. It should just close.
